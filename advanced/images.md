# Images
##### How to add images in markdown?
Lets see!

The syntax looks like this, where square brackets is _alt text_  and normal brackets is _URL_ ![]()

```
![Alt text](https://url.com/example.png)
```

**Tip**: You can put all your images in a special repo like:
```
https://github.com/UserName/special-image-repo/raw/branchName/image1.png
```
