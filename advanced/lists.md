# Lists in markdown
In markdown, there are two main ways to create listes, unordered and ordered listes.
## Unordered list
For unordered lists, you can use `+`, `-` or `*` to create an ordered list
* Asteriks
+ Plus sign
- Dash

## Ordered list
You can type `<number>` and `.` together to create an ordered list
1. Apple
2. Pear
3. Lemon
